Link to the Future

A heatmap for Seattle Link stations.

About

Who?

The project was made by Preston, Preston, Alex, and Quan for MapHacks@UW 2018.

What?

A web app visualizing the ideal locations for new Link Light Rail stations in Seattle.

When?

This analysis was conducted in Spring 2018.

Why?

Seattle is in gridlock! Surface transit has become crippling for cars, buses, and bicycles alike. We need a grade-separated Link to the future!

How?

A location allocation analysis for stations based on job and population density, as well as existing Link and bus facilities. No attempt was made to analyze routing or feasibility.
